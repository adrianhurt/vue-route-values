# Installation

yarn add vue-route-values

or

npm install vue-route-values --save

And that's all! Here you have a simple usage example.

<template>
    <Value #default="{ value, set }">
        ...
    </Value>
</template>

<script>
import { Value } from 'vue-route-values'

export default {
    name: 'ValueExample',
    components: { Value },
}
</script>
